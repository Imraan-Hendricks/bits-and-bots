# Bits And Bots

This project demonstrates generating and displaying 20 lines of text on a webpage based on specified conditions using JavaScript.

## Usage

1. Open `index.html` in the `src` directory using Google Chrome browser.

   ```bash
   cd src
   open index.html
   ```

## Project Structure

- `src/index.html`: HTML file displaying the output on the webpage.
- `src/assets/scripts.js`: JavaScript code responsible for generating the output.
- `src/assets/styles.css`: CSS styles for the webpage.
- `.editorconfig`: EditorConfig file for consistent code formatting.
- `.vscode/settings.json`: Visual Studio Code settings for the project.
- `.prettierrc.js`: Prettier configuration for code formatting.

## Instructions

Open the `index.html` file in a web browser to view the output of the functions generated by the JavaScript code in `scripts.js`.
